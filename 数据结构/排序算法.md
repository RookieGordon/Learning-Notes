# 排序的分类

|  排序策略  |  算法         |
|:-------|:------------|
|  交换    |  冒泡排序，快速排序  |
|  分配    |  计数排序，基数排序  |
|  选择    |  选择排序，堆排序   |
|  归并    |  归并排序       |
|  插入    |  插入排序，希尔排序  |     
# 冒泡排序

基于交换的排序。每轮查找序列中的最大值，然后将其放到序列尾部。
