# 三角形绘制算法

给出3个点$v_{0}, v_{1}, v_{2}$，绘制出三角形：
1. 找到本三角形的最小包围盒范围
2. 遍历包围盒内的像素，使用叉乘判断是否在三角形内
3. 三角形内所有像素组成集合返回

## 三角形最小包围盒

依次判断3个点的x分量最小值和最大值，y分量最小值和最大值。
